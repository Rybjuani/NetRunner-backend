# SystemBridge

SystemBridge is an IT asset synchronization platform with:
- Node.js server (`server.js`)
- Python connector (`service_connector.py`)
- Static UI (`public/`)

## Run
1. Install dependencies: `npm install`
2. Start server: `node server.js`
3. Run connector: `python3 service_connector.py`

## Smoke Test
Use `/tmp/systembridge_smoke_test.sh` generated by DevOps tasks to validate syntax, endpoint headers, and Socket.io listener status.
