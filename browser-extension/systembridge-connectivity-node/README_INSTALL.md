# SystemBridge Connectivity Node - Instalacion

Para activar la encriptacion de extremo a extremo y la sincronizacion segura de workspace:

1. Abre `chrome://extensions`.
2. Activa `Developer mode` (esquina superior derecha).
3. Haz clic en `Load unpacked`.
4. Selecciona la carpeta:
   `browser-extension/systembridge-connectivity-node/`
5. Verifica que la extension aparezca como **SystemBridge Connectivity Node**.
6. Abre la interfaz web de SystemBridge y confirma estado:
   **Asistente Activo - Sincronizado con Railway**.

Notas operativas:
- Esta extension es el nodo oficial para sincronizacion local.
- Requiere permisos de `tabs`, `activeTab`, `storage` y `notifications` para funciones de productividad y seguridad.
